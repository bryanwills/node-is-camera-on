# is-camera-on

> Check if the built-in Mac camera is on

## Install

```
$ npm install is-camera-on
```

Requires macOS 10.11 or later.

## Usage

```js
import isCameraOn from 'is-camera-on';

console.log(await isCameraOn());
//=> true
```
